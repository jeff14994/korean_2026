# 韓語助詞學習教材 Korean Grammar Study Tool

一個互動式韓語助詞學習網頁，包含例句播放、閱讀理解測驗、單字發音等功能。

## 功能總覽

### Page 1：助詞整理
- **23 組助詞例句**，涵蓋主格、受格、對象、連接、地點、時間、比喻等用法
- 每個句子都有**獨立播放按鈕**，方便逐句練習
- **單字解釋**列出重點韓語單字與中文意思，點擊韓語單字可直接聽發音
- **文法重點**分頁整理所有助詞的核心用法

### Page 2：閱讀理解
- **22 道選擇題**（含範例），讀完短文後判斷主題
- 即時計分與作答統計
- 每題選項下方有**發音預覽按鈕**，可先聽選項韓語發音再作答
- 答題後顯示正確答案與解說

### 共用功能
- 🔊 **雙語音模式**：線上 Google TTS（較自然）或瀏覽器內建語音
- 🎚️ **播放速度調整**（0.7x ~ 1.1x）
- 🙈 **隱藏/顯示中文**：隱藏翻譯與文法說明，保留韓語單字方便自測
- 🌙 **深色/淺色模式**，自動記住偏好
- 📱 **響應式設計**，支援手機與桌面瀏覽

## 使用方式

直接用瀏覽器開啟 `main.html` 即可使用，無需安裝任何套件。

```bash
open main.html
```

## UI/UX 改善紀錄

### v1.0 — 初始版本
- 基本助詞例句展示與 TTS 播放

### v1.1 — 互動與體驗優化

| 改善項目 | 說明 |
|---------|------|
| **每句獨立播放按鈕** | 同一編號下的多個句子各自有播放按鈕，不再共用，方便逐句練習 |
| **單字點擊播放** | `.word-meanings` 中的粗體韓語單字可直接點擊聽發音 |
| **隱藏中文時保留單字** | 切換「隱藏中文」後，單字區域仍顯示韓語詞彙（隱藏中文翻譯部分），方便自我測試 |
| **浮動控制按鈕** | 右下角固定顯示「隱藏中文」與「深色模式」按鈕，滾動到任何位置都能快速切換 |
| **頁面切換記住捲動位置** | 在 Page 1 和 Page 2 之間切換時，自動記住並恢復上次的閱讀位置 |
| **Page 2 題號標示** | 播放按鈕顯示對應題號（如「▶ 播放句子 보기」、「▶ 播放句子 1」） |
| **選項發音預覽** | Page 2 每題選項下方新增「🔊 點擊聽發音」區域，可先聽韓語發音再作答，不會觸發答題 |
| **防止誤選文字** | 單字與發音按鈕加入 `user-select: none`，點擊時不會意外選取文字 |
| **事件委派機制** | 使用 event delegation 確保動態修改 DOM 後事件監聯器仍然有效 |
| **15일/18일 翻譯補充** | Page 2 第 5 題補充 15일＝15號、18일＝18號 的單字翻譯 |

### v1.2 — Bug Fixes (2026-05-13)

| 修復項目 | 說明 |
|---------|------|
| **深色模式卡片標題不可見** | `.card-title` 原本硬編碼為 `#111827`（深色），在深色模式下幾乎看不到。改為 `var(--text)` 自動適應主題 |
| **Google TTS 在 http:// 無法播放** | 透過 `http://127.0.0.1:8000` 存取時，瀏覽器會帶 `Referer` 標頭，Google 會拒絕請求。加入 `<meta name="referrer" content="no-referrer">` 移除 Referer，使 http:// 與 file:// 行為一致 |
| **GitHub Pages 相容性** | 同上修復，`no-referrer` meta 標籤也解決了 GitHub Pages 上 Google TTS 被 CORS 阻擋的問題 |
| **浮動頁面切換 (◀ 1/2 ▶)** | 右下角新增 prev/next 箭頭導航，顯示「頁碼 / 總頁數」，可擴展至任意數量頁面 |
| **長按跳頁選單** | 長按頁碼指示器（400ms）彈出頁面列表，顯示每頁標題，點擊即可跳轉，點擊外部自動關閉 |

## 技術細節

- 純 HTML/CSS/JavaScript，無框架依賴
- TTS 優先使用 Google Translate TTS API，失敗時自動降級為 Web Speech API
- 音訊快取機制避免重複載入
- CSS 變數實現深色/淺色主題切換
- `localStorage` 記住主題偏好
